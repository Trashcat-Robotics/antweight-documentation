1.4 AUTO_EXE-0005

AUTO_EXE shall perform the following steps during system disarm behavior:

1) Set the autonomy mode to SYS_DISARM

2) Command WEAP_MGR to safe disarm

3) Command DRIVE_MGR to safe disarm

4) Wait for a response from WEAP_MGR confirming disarm

5) Wait for a response from DRIVE_MGR confirming disarm

6) Set the autonomy mode to OFF

Parent links: SW-0006

1.0 WEAP_MGR-0001

WEAP_MGR shall command the autonomy LED indicator to be yellow when weapon mode is IDLE, and purple otherwise.

1.2 WEAP_MGR-0003

WEAP_MGR shall provide an interface to safely disarm the active weapon via the following steps:

1) Set the WEAP_MGR mode to DISARM

2) Rotate the motor backwards (slowly) until a sensor indicates the weapon is deployed

3) Command PWR_MGR to open the switch to the weapon motor

4) Set WEAP_MGR mode to IDLE

Parent links: SW-0007

Attribute Value
verified-by unverified
verified-date unverified
verified-comment unverified
type Requirement
rationale If the spring is completely or semi-engaged on power on, want to slowly remove tension. Rotating forward would cause the flipper to snap dangerously. Instead rotating backwards will alleviate the pressure.
